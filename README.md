# Responsible AI RAG Assistant

## AI Governance, GDPR, and EU AI Act Knowledge Support

This project builds a Retrieval-Augmented Generation assistant that answers questions from selected public documents about Responsible AI, GDPR, EU AI Act concepts, transparency, fairness, explainability, human oversight, and AI governance documentation.

The assistant is designed as a portfolio-ready AI Engineering project. It demonstrates document ingestion, text chunking, embeddings, vector search, source-grounded generation, responsible AI guardrails, evaluation, and a simple user-facing app.

## Project Status

In development.

## Important Disclaimer

This project is for educational and portfolio purposes only.

It does not provide legal advice. Regulatory and compliance questions should always be reviewed by qualified legal, privacy, or compliance professionals.

## Planned Features

* Public document ingestion
* Text cleaning and chunking
* Embedding generation
* Vector database search
* RAG-based question answering
* Retrieved source display
* Responsible AI guardrails
* Evaluation dataset
* Retrieval quality evaluation
* Answer faithfulness evaluation
* Failure-case analysis
* Streamlit app
